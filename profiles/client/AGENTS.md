# Client engagement — confined workspace

This is a CONFINED workspace for an external engagement. You are in a bubblewrap namespace: you see
ONLY this directory and your own config. The rest of the host is not mounted.

- Everything you produce goes in `deliverables/`. Inputs are in `data/`.
- Do not assume access to anything outside this workspace — there is nothing there.
- Treat all material here as confidential to this engagement. It never mixes with other work
  (the confinement guarantees that structurally, and the second brain never reads confined content).
- Read `WORKSPACE.md` for this engagement's specific scope and standing rules.
